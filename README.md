## 📖 About

**MR-CAPSULES** is a non-profit learning platform established in August 2025.

What began as simple HTML files shared among friends has evolved into a comprehensive web-based educational platform. Initially created to archive and distribute previous years' study materials, the project has grown into a centralized repository containing lecture notes, question banks, faculty presentation slides, and various academic resources.

More than just a resource archive, MR-CAPSULES aims to be a digital sanctuary where students can learn, collaborate, and grow together through shared knowledge.

🌐 **Website:** https://mr-capsules.vercel.app

---

## 🎯 Mission

To build an accessible academic ecosystem where educational resources can be preserved, shared, improved, and passed on to future generations of students.

## 🤝 Contributing

MR-CAPSULES is built through community collaboration. We welcome students and contributors who share the vision of making educational resources more accessible.

### 📚 Content Division

Responsible for collecting, organizing, and maintaining learning resources.

**Preferred Skills**
- Academic resource management
- Documentation and organization
- Attention to detail

---

### ✅ Correction Division

Responsible for reviewing question banks, answer keys, explanations, and academic materials.

**Preferred Skills**
- Strong academic understanding
- Critical thinking
- Error detection and verification
- Ability to validate answers and references

---

### 💻 Developer Division

Responsible for platform development, maintenance, optimization, and feature implementation.

**Preferred Skills**
- HTML, CSS, JavaScript
- Git & GitHub
- Frontend development
- UI/UX fundamentals
- Problem-solving mindset
- Willingness to learn and collaborate

Beginners are welcome. Curiosity, consistency, and a willingness to learn are valued more than experience.
